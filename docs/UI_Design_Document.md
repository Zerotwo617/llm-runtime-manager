# LLM Runtime Manager UI 设计规范文档（Apple 极简风）

## 1. 设计目标
- Apple极简风格
- 高信息密度但清晰
- 支持明暗模式
- 系统级AI工具UI

## 2. 布局结构
- 左侧Sidebar
- 顶部Topbar
- 中间Dashboard
- 底部性能监控

## 3. Sidebar
- 240px
- 仪表盘/模型/参数/环境/运行/监控/日志/设置

## 4. Dashboard
- CPU / RAM / GPU / 存储 / 网络卡片
- 模型快速启动区
- 参数推荐区
- 环境状态区

## 5. 设计风格
- 圆角 12-16px
- 毛玻璃 blur 16px
- 低饱和蓝紫色
- Apple-like spacing

## 6. 颜色系统

Light:
- bg #F5F6F8
- card #FFFFFF
- primary #3B82F6

Dark:
- bg #0B0F17
- card #151A23
- primary #60A5FA

## 7. 字体
- SF Pro / Inter / PingFang SC

## 8. 动效
- 150–250ms ease-out
- fade / slide / hover lift

## 9. 核心原则
- 信息分层清晰
- 不堆视觉
- 控制感强
